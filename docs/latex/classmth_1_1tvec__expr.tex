\subsubsection*{template$<$typename Derived, bool memoize$>$\newline
class mth\+::tvec\+\_\+expr$<$ Derived, memoize $>$}

The base vector expression class, used for the C\+R\+TP pattern. 

This base class contains almost all of the operations on vectors, such as detailed component access and arithmetic operator overloads, equality operator overloads. However, it does not provide iterator member functions, those are only accessible from a concrete {\ttfamily \mbox{\hyperlink{classmth_1_1tvec}{mth\+::tvec}}} value.
