\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Func, typename... Vecs$>$\newline
class mth\+::tvec\+\_\+map$<$ Func, Vecs $>$}

Vector expression for mapping a function over the components of a list of vectors. 

This is the functional concept of mapping; a function f mapped over vectors {\ttfamily a = \{a0, a1, ...\}, b = \{b0, b1, ...\}, c = \{c0, c1, ...\}} is essentially the vector {\ttfamily \{f(a0, b0, c0), f(a1, b1, c1), ...\}}.

The pack of vectors to map over must be non-\/empty.
