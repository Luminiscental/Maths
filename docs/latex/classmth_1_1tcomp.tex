\subsubsection*{template$<$typename T$>$\newline
class mth\+::tcomp$<$ T $>$}

Complex number expression for a concrete / evaluated complex value. 

Various aliases for this type are provided with the typical type prefixes and default types\+: {\ttfamily icomp} for {\ttfamily tcomp$<$int$>$}, {\ttfamily dcomp} for {\ttfamily tcomp$<$double$>$}, {\ttfamily ucomp} for {\ttfamily tcomp$<$unsigned int$>$}, {\ttfamily fcomp} for {\ttfamily tcomp$<$float$>$}. Leaving out the type prefix defaults to float, so {\ttfamily comp} is {\ttfamily fcomp};
