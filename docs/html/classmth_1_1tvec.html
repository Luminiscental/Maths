Detailed Description

template<typename T, size_t N>
class mth::tvec< T, N >

Vector expression for a concrete / evaluated vector value.

This implements the typical iterable protocol to allow range-for loops, whereas the generic mth::tvec_expr doesn't; only concrete / evaluated vectors can be iterated. The iterators are forwarded directly from std::array for the components.

Various aliases for this type are provided with the typical type prefixes and default types: ivec1 for tvec<int, 1>, dvec3 for tvec<double, 3>, uvec2 for tvec<unsigned int, 2>, fvec4 for tvec<float, 4>. Leaving out the type prefix defaults to float, so vec3 is fvec3 and so on.
