Detailed Description

template<typename T>
class mth::tcomp< T >

Complex number expression for a concrete / evaluated complex value.

Various aliases for this type are provided with the typical type prefixes and default types: icomp for tcomp<int>, dcomp for tcomp<double>, ucomp for tcomp<unsigned int>, fcomp for tcomp<float>. Leaving out the type prefix defaults to float, so comp is fcomp;

Template Parameters
TThe scalar type for the real and imaginary part of the value.
