mth - A Maths library

mth is a maths library built around expression template classes to work with objects such as vectors, matrices and quaternions, with helper classes to deal with solving equations. No guarantees of stability or quality so be warned.

Building:

To build from source, you will need cmake and a c++ compiler supporting c++17.

First, clone the repo and run cmake:

$ git clone https://github.com/Luminiscental/mth
$ cd mth && mkdir build && cd build
$ cmake .. # Optionally pass -DTESTS=ON to run tests when building

Then run the generated build files, on mac/linux there should be a Makefile in the build directory that you can run with make. On windows there should be a Visual Studio solution file mth.sln; open this and build the solution.

After building there should be a libmth.a or mth.lib file for linking into programs that use mth.
