Detailed Description

template<typename Derived, bool memoize>
class mth::tvec_expr< Derived, memoize >

The base vector expression class, used for the CRTP pattern.

This base class contains almost all of the operations on vectors, such as detailed component access and arithmetic operator overloads, equality operator overloads. However, it does not provide iterator member functions, those are only accessible from a concrete mth::tvec value.

Template Parameters
DerivedThe type of the class deriving from this base for CRTP.
memoizeBool for whether the base class should handle memoizing operator[].

Member function specializing mth::vec::map to the single parameter case.

Maps a function over the components of this vector, in the functional sense. So for a vector v = {v0, v1, ...}, v.map(f) = {f(v0), f(v1), ...} essentially.
